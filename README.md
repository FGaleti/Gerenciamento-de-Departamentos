# API de Gerenciamento de Departamentos

API REST desenvolvida com **Spring Boot** para o gerenciamento de departamentos em um contexto organizacional. O foco deste projeto é a prática de mapeamento JPA de entidades simples e a estruturação de serviços e controladores para operações CRUD.

## 🚀 Tecnologias Utilizadas
* Java
* Spring Boot
* Spring Data JPA
* Banco de Dados Relacional

## 📌 Endpoints da API

A API expõe os seguintes endpoints na rota base `/departamentos`:

* `POST /departamentos`: Cria um novo departamento.
* `GET /departamentos`: Lista todos os departamentos cadastrados.
* `GET /departamentos/{id}`: Busca um departamento específico pelo seu ID.
* `DELETE /departamentos/{id}`: Deleta um departamento específico pelo seu ID.
